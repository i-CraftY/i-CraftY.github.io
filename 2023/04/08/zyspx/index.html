零、前言

1.中药商品学研究的中心内容是

中药商品质量

一、中药商品学概述

1. 商品的含义

商品是用来交换的劳动产品。

2. 商品具有的基本特征

  1. 具有使用价值的劳动产品

  2. 供社会消费的劳动产品

  3. 必须通过交换才能到达他人手中的劳动产品

3. 商品的基本属性

  1. 使用价值:是指商品能够以自身自然属性来满足人们某种需要的属性

  2. 价值:是凝结在商品中的抽象劳动

4. 中药商品学定义

中药商品学是一门研究商品、保健品和其他中医医疗用品作为商品的使用价值及在流通过程中实现价值的规律的应用科学。

5.保健食品定义

是指具有特定保健功能或者以补充维生素、矿物质为目的的食品

6.中药化妆品定义：

是指以中医药理论为指导，含有中药，具有明显的功能性，针对性强，以预防为主，以涂抹、喷洒或者其他类似方法，散布于人体表面的任何部位，以达到清洁、保养、美容、修饰和改变外观，或者修正人体气味，保持良好状态为目的的化学工业品或精细化工产品

7.有形附加物：如商品名、企业名、包装及标记、商标、专利标记等

8.现存最早的中药商品学专著当推《范子计然》

9.GAP、GLP、GMP、GCP、GSP

GAP(Good Agriculture Practice)：药材生产管理规范

GLP(Good Laboratory Practice)：药物非临床研究质量管理规范

GMP(Good Manufacturing Practice)：药品生产质量管理规范

GCP(Good Clinical Practice)：药物临床试验质量管理规范

GSP(Good Supply Practice)：药物经营质量管理规范

二、中药商品的分类、品种与编码

1.处方药定义

是指只能通过具有执照的医师或者有他们的处方才能调配，并在医务人员的指导下应用的药物

2.非处方药定义

是指那些只要消费者按照药品标签上列出的规定就能安全使用的药物

3. 非处方药（OTC）的分类

相对来说乙类的安全性更好，而甲类的安全性需要进一步验证。

甲类，必须在药店出售。乙类，除药店外，还可在药监部门批准的宾馆、商店等商业企业中零售。

4. 根据生产地分类

批准文号

国药准字H 20 00 0655

H:化学药

20:批准文号来源。10原卫生部，19和20国家药监局，其他各省、自治区、直辖市的行政代码

00:2000年，公元年号后两位

0655:顺序号

1.国产药品

Z代表中成药，S代表生物制品，J代表进口药品国内分包装，H代表化学药品

  1. 进口药品

不仅要标注批准文号”国药准字J“字样

a.港澳台还要标明”医药产品注册证号“

b.其他国家进口还要标明”进口药品注册证号“

5.按医药商品的特殊性分类

麻醉药品：连续使用后易产生身体依赖性、能成瘾的药品。

精神药品：直接作用于中枢神经系统，使之兴奋或抑制，连续使用能产生依赖性的药品。

毒性药品：毒性剧烈、治疗量与中毒剂量相近，使用不当会致人中毒或死亡的药品。

放射性药品：用于临床诊断或治疗的放射性核制剂或者其他标记药品。

三、中药商品的质量与标准

1. 质量的定义

一组固有特性满足要求的程度。

2. 中药商品的质量特征

有效性(基本特征)：指在规定的适应症、用法和用量条件下能满足预防、治疗、诊断人的疾病，有目的地调节人的生理机能。

安全性(基本特征):是指中药商品在按规定的适应症、用法和用量使用的情况下,对使用者生命安全的影响程度。

稳定性(重要特征):是指中药商品在规定的条件下保持其有效性和安全性的能力。规定的条件包括中药商品的有效期限以及医药商品生产、贮存、运输和使用的要求。

均一性(重要特征):是指中药商品的每一单位产品都符合有效性、安全性的规定要求。

经济性:指中药商品生产、流通过程中形成的价格水平。

3.总灰分、酸不溶灰分

总灰分：是指中药材本身经过灰化后遗留的不挥发性的无机成分，以及中药材表面附着的不挥发性无机成分

酸不溶灰分：是指总灰分中加入10%盐酸处理，得到不溶于10%盐酸的灰分

4. 我国中药商品的二级标准

(1)国家标准 — 《中华人民共和国药典》

(2)局颁标准

四、中药商品的市场

  1. 市场的概念

    市场是指对某种商品有需求和有购买能力的顾客或用户，是显在顾客和潜在顾客的总和

  2. 医药市场类型

    1. 按医药市场规模分类
    2. 按医药市场客体组成分类

    4. 按商品分类管理要求分类
    5. 按医药产品种类分类

  3. 四大药都

    安徽亳州、河北安国、成都荷花池、河南禹州

  4. 国内主要药材市场

  5. 全国中药材指数

    成都指数反映中药材价格和行业景气度的重要指标

    康美·中国中药材价格指数：全国性药材价格指数

五、中药商品的物流管理

  1. 公路运输

​ 优点：速度较快、可靠性高和对产品损伤较小

​ 缺点：变动成本高

  1. 铁路运输

    优点：变动成本较低

    缺点：灵活性不高

  2. 水路运输

    优点：能够运输数量巨大的货物适合运输低价值的货物

    缺点：营运范围和运输速度受到限制

  3. 航空运输

    优点：运输速度快，固定成本低

    缺点：变动成本高

  4. 管道运输

    中药商品很少用管道运输

六、中药商品的储存与养护

1.中药商品储存与养护的意义

  1. 确保中药商品安全有效

  2. 加强中药商品流通

  3. 监督中药质量

  4. 降低流通费用

  5. 中药商品养护有助于维护社会财富

2. 中药分区分类储存应遵守的原则

合格品为绿色

不合格品为红色

待验品为黄色

3. 中药商品养护的原则

(1)贯彻”预防为主“原则

(2)遵循先产先出、易变先出、近期先出的原则

​ 先产先出：是指同一商品，对先生产的批号应先出库

​ 易变先出：是指同一商品，对不宜久储易于变质的应先出库

​ 近期先出：即近失效期先出库

(3)在库中药定期检查原则

  1. 逐日检查

  2. 定期检查

  3. 突击检查

    检查内容：库房内的温度、湿度，中药的外观性状和质量变化情况，中药包装的变异，中药药垛安全，中药储存动态

4. 温度、湿度是影响中药变质的重要因素

冷处：2~10℃

凉限处：20℃以下

室温温度控制在10-30℃；相对湿度控制在60%-75%。

5. 影响中药商品质量的因素

内因(主导作用)：药品本身的物理、化学性质的变化

外因(决定作用)：温度、湿度、空气、日光、储存时间、虫害、霉菌及包装

6. 有效期：指中药在规定的储存条件下，能够保持质量合格的期限，要求使用单位在规定期限内使用。

7. 室温(10-30℃)

8. ”阴凉处“或”凉限处“是指不超过20℃

9. 冷处是指2-10℃

10.泛油又称“走油”

是指含挥发油、油脂、糖类等中药材，在一定的温度和湿度情况下，出现油脂外溢、质地变软、发黏、颜色变深，散发出败油气味等现象。

七、中药商品的包装

1. 商品包装的概念

为了在流通过程中保护产品，方便储存，促进销售，按一定技术方法而采用的容器、材料及辅助等的总体名称。

中药商品包装也是中药商品品质的重要组成部分。

2. 中药商品包装的作用

(1)保护中药商品

保护中药商品是中药包装最基本的作用

(2)便于调配与流通

(3)美化装潢、促进销售

(4)方便消费

3. 中药商品包装材料是指用来包装中药商品的包装材料。它是可服用的、接触中药品的，或作功能性外包装的包装材料和包装辅助材料的总称。

4. 中药商品包装的材料

(1)纸质材料

优点

  1. 成本低廉、重量较轻、加工性能好、便于成型
  2. 易于印刷
  3. 满足各类包装需求
  4. 不产生污染，可以回收
  5. 可复合
  6. 品种多样

缺点:耐用性差，强度较低

(2)塑料材料

优点

  1. 强度好
  2. 阻隔性好
  3. 质轻携带方便
  4. 透明

缺点:难以降解，易污染环境，透气性差

(3)玻璃包装

优点

  1. 可长期储存药品
  2. 光洁，造型美观
  3. 复用、降低成本

缺点:重、质脆、易碎

(4)铝箔

无毒无味，量轻，遮光性好，防潮、防挥发，保香性能优良，形状稳定

5.中药商品包装用材料的管理

药包材须经药品监督管理部门注册并获得《药包材注册证书》后方可生产。

《药包材注册证》统一编号格式:

I类产品编号为:
国药包字XXXXXXXX
II、III类产品编号为:
X (省、自治区和直辖市的简称)药包字XXXXX
《进口药包材注册证》编号为:
JXXXXXXXX
(J为进口代号，前四位数字为年号，后四位数字年内顺序号)。

6.中药商品包装的分类

按形态不同分类

个包装：中药商品个别加以包装，也称为小包装（销售包装）

内包装：一般多指直接接触药品的包装

外包装：将内包装好了的中药商品装入箱、袋、盒、罐等容器中后，再加上一层的包装

7.药品的标签

药品的内标签至少应当标注药品通用名称、规格、产品批号、有效期等内容。

治疗用生物制品有效期的标注自分装日期计算，其他药品有效期的标注自生产日期计算。

有效期若标注到日，应当为起算日期对应年月日的前一天，若标注到月，应当为起算月份对应年月的前一月。

八、中药商品的广告

违法广告分析

1.广告的定义

广告是为了某种特定的需要，通过一定形式的媒体，公开而广泛地向公众传递信息的宣称手段。

  1. 有偿

  2. 非个体传播

  3. 劝服性

  4. 目的性、计划性、连续性

2.广告的作用

(1)刺激需求，推动购买

(2)利于竞争，促进生产与经营

(3)促进和支援了企业的人员促销

(4)有助于企业形象的树立

(5)促进国际贸易交往

(6)普及健康知识，培养人们正确的生活方式和美好情操

3.药品广告批准文号

X药广审(视)第0000000000号、X药广审(声)第0000000000号、X药广审(文)第0000000000号

"X"为各省、自治区、直辖市的简称。

"0"由10位数字组成，前6位代表审查年月，后4位代表广告批准序号。

”视“、”声“、”文“代表用于广告媒介形式的分类代号。

九、中药专利和商标

  1. 中药专利的授予原则

对于同一个发明只能授予一个专利权

  1. 商标

    商标是用文字、符号、图形综合组成，附标在商品上，作为该商品与其他同类商品区别的一种标志。

  2. 商标的特征

    • 商标是商品上的标志

      使用商标的商品必须同时具备以下三个条件：

      1. 必须是以一定物质形态存在的实体性商品，非物质形态的服务性商品，不能注册商标
      2. 必须是动产物
      3. 必须是能进行批量生产和反复交易的流通物

    • ⭐商品是一种区别标志

  3. 商标的作用

    • 区别作用：这是商标最重要、最本质的功能
    • 保证作用
    • 增值作用
    • 宣传作用
    • 促进作用

  4. 商标继承权：商标注册权如属于某个家庭或个人，当商标所有人死亡或丧失民事能力时，其继承人可以依法继承被继承人商标权。

  5. 使用注册商标应当标明“注册商标”字样或标明注册标记“注”或"“®️”。

  6. 人用药品和烟草制品必须使用注册商标

  7. 注册商标的续展期为12个月(1年)
