一、中药商品学概述

1. 商品的含义：商品是用来交换的劳动产品。

2. 商品具有的基本特征

  1. 具有使用价值的劳动产品

  2. 供社会消费的劳动产品

  3. 必须通过交换才能到达他人手中的劳动产品

3. 商品的基本属性

  1. 使用价值

  2. 价值

4. 中药商品学定义：中药商品学是一门研究商品、保健品和其他中医医疗用品作为商品的使用价值及在流通过程中实现价值的规律的应用科学。

5

GMP认证：药品生产质量管理规范
GCP认证：药物临床试验质量管理规范
GSP认证：药物经营质量管理规范

二、中药商品的分类、品种与编码

1. 非处方药（OTC）的分类

相对来说乙类的安全性更好，而甲类的安全性需要进一步验证。

甲类，必须在药店出售。乙类，除药店外，还可在药监部门批准的宾馆、商店等商业企业中零售。

2. 根据产地分类

批准文号

国药准字H 20 00 0655

H:化学药

20:批准文号来源。10原卫生部，19和20国家药品监管部门，其他各省、自治区、直辖市的行政代码

00:2000年，公元年号后两位

0655:顺序号

1.国产药品

Z代表中成药，S代表生物制品，J代表进口药品国内法分包装，H代表化学药品

  1. 进口药品

不仅要标注批准文号”国药准字J“字样

a.港澳台还要标明”医药产品注册证号“

b.其他国家进口还要标明”进口药品注册证号“

3.特殊药品

麻醉药品：连续使用后易产生身体依赖性、能成瘾的药品。

精神药品：直接作用于中枢神经系统，使之兴奋或抑制，连续使用能产生依赖性的药品。

毒性药品：毒性剧烈、治疗量与中毒剂量相近，使用不当会致人中毒或死亡的药品。

放射性药品：用于临床诊断或治疗的放射性核制剂或者其他标记药品。

三、中药商品的质量与标准

1. 质量的定义

一组固有特性满足要求的程度。

2. 中药商品的质量特征

有效性(基本特征)：指在规定的适应症、用法和用量条件下能满足预防、治疗、诊断人的疾病，有目的地调节人的生理机能。

安全性(基本特征):是指中药商品在按规定的适应症、用法和用量使用的情况下,对使用者生命安全的影响程度。

稳定性(重要特征):是指中药商品在规定的条件下保持其有效性和安全性的能力。规定的条件包括中药商品的有效期限以及医药商品生产、贮存、运输和使用的要求。

均一性(重要特征):是指中药商品的每一单位产品都符合有效性、安全性的规定要求。

经济性:指中药商品生产、流通过程中形成的价格水平。

3. 标准的定义

对重复性事物和概念所做的统一规定。

4. 我国中药商品的二级标准

(1)国家标准 — 《中华人民共和国药典》

(2)局颁标准

5. 中成药质量标准的内容

没有商品名，只有通用名。

处方中药材按君臣佐世顺序排列。

处方中药材生品与炮制品写法

药材需要炮制的，用括号注明，如当归（酒炒）

性状通常是依据样品除去包装后的实际情况拟定。

六、中药商品的储存与养护

1. 中药分区分类储存应遵守的原则

合格品为绿色

不合格品为红色

待验品为黄色

2. 中药商品养护的原则

(1)贯彻”预防为主“原则

(2)遵循先产先出、易变先出、近期先出的原子

(3)在库中药定期检查原则

3. 中药商品的在库养护：是指中药商品在仓库储存过程中所进行的保养和维护工作。

4. 温度、湿度是影响中药变质的重要因素

室温温度控制在10-30℃；相对湿度控制在60%-75%。

5. 影响中药商品质量的因素

内因(主导作用)：药品本身的物理、化学性质的变化

外因(决定作用)：温度、湿度、空气、日光、储存时间、虫害、霉菌及包装

6. 有效期：指中药在规定的储存条件下，能够保持质量合格的期限，要求使用单位在规定期限内使用。

7. 室温(10-30℃)

8. ”阴凉处“或”凉阴处“或”凉限处“是指不超过20℃

9. 冷处是指2-10℃

七、中药商品的包装

1. 商品包装的概念

为了在流通过程中保护产品，方便储存，促进销售，按一定技术方法而采用的容器、材料及辅助等的总体名称。

中药商品包装也是中药商品品质的重要组成部分。

2. 中药商品包装的作用

(1)保护中药商品

保护中药商品是中药包装最基本的作用

(2)便于调配与流通

(3)美化装潢、促进销售

(4)方便消费

3. 中药商品包装材料是指用来包装中药商品的包装材料。它是可服用的、接触中药品的，或作功能性外包装的包装材料和包装辅助材料的总称。

4. 中药商品包装的材料

(1)纸质材料

优点

  1. 成本低廉、重量较轻、加工性能好、便于成型
  2. 易于印刷
  3. 满足各类包装需求
  4. 不产生污染，可以回收
  5. 可复合
  6. 品种多样

缺点:耐用性差，强度较低

(2)塑料材料

优点

  1. 强度好
  2. 阻隔性好
  3. 质轻携带方便
  4. 透明

缺点:难以降解，易污染环境，透气性差

(3)玻璃包装

优点

  1. 可长期储存药品
  2. 光洁，造型美观
  3. 复用、降低成本

缺点:重、质脆、易碎

(4)铝箔

无毒无味，量轻，遮光性好，防潮、防挥发，保香性能优良，形状稳定

5.中药商品包装用材料的管理

药包材须经药品监督管理部门注册并获得《药包材注册证书》后方可生产。

《药包材注册证》统一编号格式:

I类产品编号为:
国药包字XXXXXXXX
II、III类产品编号为:
X (省、自治区和直辖市的简称)药包字XXXXX
《进口药包材注册证》编号为:
JXXXXXXXX
(J为进口代号，前四位数字为年号，后四位数字年内顺序号)。

6.中药商品包装的分类

按形态不同分类

个包装：中药商品个别加以包装，也称为小包装（销售包装）

内包装：一半多指直接接触药品的包装

外包装：将内包装好了的中药商品装入箱、袋、盒、罐等容器中后，再加上一层的包装

7.药品的标签

药品的内标签至少应当标注药品通用名称、规格、产品批号、有效期等内容。

治疗用生物制品有效期的标注自分装日期计算，其他药品有效期的标注自生产日期计算。

有效期若标注到日，应当为起算日期对应年月日的前一天，若标注到月，应当为起算月份对应年月的前一月。

七、中药商品的广告

违法广告分析

1.广告的定义

广告是为了某种特定的需要，通过一定形式的媒体，公开而广泛地向公众传递信息的宣称手段。

  1. 有偿

  2. 非个体传播

  3. 劝服性

  4. 目的性、计划性、连续性

2.广告的作用

(1)刺激需求，推动购买

(2)利于竞争，促进生产与经营

(3)促进和支援了企业的人员促销

(4)有助于企业形象的树立

(5)促进国际贸易交往

(6)普及健康知识，培养人们正确的生活方式和美好情操

3.药品广告批准文号

X药广审(视)第0000000000号、X药广审(声)第0000000000号、X药广审(文)第0000000000号

"X"为各省、自治区、直辖市的简称。

"0"由10位数字组成，前6位代表审查年月，后4位代表广告批准序号。

”视“、”声“、”文“代表用于广告媒介形式的分类代号。
