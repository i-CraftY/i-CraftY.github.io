一、绪论

1.植物化妆品开发学

是开发含植物成分的化妆品的一门学科

2.化妆品定义

是指以涂擦、喷洒或者其他类似方法，施用于人体表面，以清洁、保护、美化、修饰为目的的日用化学工业品

3.植物化妆品

主要是由含有植物源成分的各种原料，经配方加工混合，不需要经过化学反应而制成的一种复杂的混合物

4.我国在法规层面不存在“药妆”的概念

化妆品是日常生活用品，植物源成分在化妆品和药品中的功效作用不同，含植物来源等中药的化妆品不能称中药化妆品

5.特殊化妆品

用于染发、烫发、防脱发、祛斑美白、防晒的化妆品及宣称新功效的化妆品

国药特字

6.按化妆品的特殊性分类

  • 普通化妆品

  • 特殊化妆品

7.化妆品特性

  1. 安全性（首要特性）

  2. 稳定性

  3. 使用性

  4. 功效性

8.植物化妆品的发展趋势

  1. 随市场需要而变化

  2. 功效宣称更规范

  3. 生产趋于自动化

  4. 新技术提升化妆品品质

二、皮肤解剖与生理功能

1.皮肤是人体最大的器官

2.角质层细胞由5~10层已经死亡的细胞组成

三、植物化妆品原料

1.基质原料与辅助原料之间没有绝对的界线

2.植物化妆品原料必须具备以下条件

  1. 不对皮肤产生刺激和毒性

  2. 不妨碍皮肤的生理作用

  3. 不会使皮肤产生异常生理变化

  4. 不促进微生物的生长和繁殖

  5. 稳定、不变色，不会产生不愉快气味和发臭

3.基质原料

基质原料是化妆品的一类主体原料，在化妆品配方中占有较大比例

4.油质原料

  1. 屏障作用

  2. 滋润作用

  3. 清洁作用

  4. 溶剂作用

  5. 乳化作用

  6. 固化作用

油脂

  1. 橄榄油：易被皮肤吸收，是较好的润肤剂

  2. 杏仁油：最不油腻的油脂

  3. 椰子油：主要成分为月桂酸、豆蔻酸的甘油酯，是制作肥皂不可缺少的油脂原料

  4. 乳木果油：安全性好，无油腻感

蜡类:动植物蜡可起到增加膏体稳定性、调节黏度、提高液体油脂的熔点、改善产品使用感和滋润、柔然皮肤的作用

  1. 羊毛脂：略有臭味

  2. 霍霍巴油：无油腻感

烃类

  1. 液体石蜡：又称白油

  2. 地蜡

  3. 微晶蜡

高级脂肪酸、高级脂肪醇和脂类

  1. 脂肪酸

    1. 月桂酸：主要用于生产香皂，各种表面活性剂
    2. 肉豆蔻酸
    3. 棕榈酸
    4. 硬脂酸

  2. 脂肪醇

    1. 月桂醇：是制备表面活性剂的重要原料

  3. 脂肪酸酯：黏度低、延展性好

    1. 肉豆蔻酸异丙酯
    2. 异硬脂酸异丙酯

  4. 合成油脂原料

5.表面活性剂

非离子型表面活性剂：乳化剂、稳泡剂、低泡去污剂

  1. 聚氧乙烯型

  2. 多元醇型

阴离子型表面活性剂

  1. 高级脂肪酸盐

  2. 磺酸盐

  3. 硫酸酯盐

  4. 磷酸酯盐

阳离子型表面活性剂：主要用作头发调理剂和杀菌剂

  1. 季铵盐类阳离子表面活性剂

  2. 铵盐类阳离子型表面活性剂

两性离子型表面活性剂

甜菜碱型两性表面活性剂

6.胶质原料

胶质原料是水溶性的高分子化合物

包括聚乙烯醇、聚乙烯吡咯烷酮、羧甲基纤维素钠、丙烯酸聚合物

作为粘胶剂、增稠剂、成膜剂、乳化稳定剂

7.溶剂原料

  1. 乙醇在化妆品中主要用作溶剂

  2. 丙二醇是化妆品中常用的保湿剂

8.香料和香料是第一过敏原

9.157项准用着色剂

10.防腐剂包括：对羟基苯甲酸酯类(尼泊金甲酯）、咪唑烷基脉、溴硝丙二酸。常用的抗氧化剂有：没食子酸

11.螯合剂

EDTA、（磷酸、柠檬酸）、抗坏血酸

12.保湿剂

保持、补充皮肤角质层中的水分，赋予皮肤和制品本身以吸湿力

脂肪醇类

  1. 甘油，又称丙三醇，是性能良好的保湿剂（价格便宜）

  2. 山梨醇

  3. 聚乙二醇（PEG）

脂肪酸及盐类

  1. 乳酸

  2. 吡咯烷酮羧酸钠（PCA-Na）：效能与透明质酸相当

  3. 透明质酸，是真皮中主要保湿成分，可保留比自身中500~1000倍的水

13.神经酰胺，延缓皮肤衰老

14.防晒剂

  1. 紫外线屏蔽剂

    • 二氧化钛
    • 氧化锌

  2. 紫外线吸收剂

    • 对氨基苯甲酸及其酯类
    • 邻氨基苯甲酸酯类
    • 三嗪类

15.植物功效性成分

  • 种类少

  • 天然化妆品原料研究主要集中在保湿和美白功效

  • 以根和茎类中药研究为主

16.常见植物功效性成分

  1. 黄酮：抗氧化性

  2. 常见植物提取物

    • 库拉索芦荟提取物：保湿

    • 金缕梅提取物：抗敏

五、各类植物化妆品设计要点

1.皮肤清洁类化妆品（游离脂肪酸、角鲨烯、角鲨烷、蜡类、胆固醇类、酯类）

分为表面活性剂型和溶剂型（卸妆油）

清洁霜：分为无水型清洁霜、乳化型清洁霜

  1. 无水型清洁霜

  2. W/O型清洁霜

    根据乳化方式的不同，可以分为蜂蜡-硼砂乳化体系和非反应式乳化体系

    2021年我国已将硼砂列为化妆品禁用原料，因此蜂蜡-硼砂乳化体系已成为历史

    将油相加热至90摄氏度灭菌并熔化，将水相加热至相同温度，将水相缓缓加入油相中，边加边搅拌，方向相同，冷却到45摄氏度，加入防腐剂和香精

洗面奶：皂基洗面奶（反应式乳化体系）

去死皮膏

去死皮制剂：聚乙烯醇（PVA），果酸、水杨酸

沐浴用品

沐浴液

组分 功能
主要表面活性剂 起泡、清洁作用
辅助表面活性剂 增泡、降低刺激性
增泡剂 增泡、稳泡、改善泡沫的质地
黏度调节剂 调节黏度
珠光剂 产生珠光外观
香精 赋香
稳定剂 防腐、抗氧化、螯合
去离子水 溶剂、稀释剂

2.面膜类化妆品

剥离面膜

  1. 成膜剂

  2. 粉剂：清洁

  3. 保湿剂：保湿

  4. 油性成分

  5. 醇类：调整蒸发速度

  6. 增塑剂：增加膜的塑性，常用的有甘油

  7. 防腐剂：抑制微生物生长，常用的为尼泊金酯类

  8. 表面活性剂：增溶作用

3.护肤类化妆品

给皮肤补充水分、营养滋润

雪花膏:一种以硬脂酸为主要油分的O/W型乳化膏霜

  1. 硬脂酸

    • 滋润
    • 固化
    • 和碱皂化生成硬脂酸盐作为乳化剂

  2. 碱类：和硬脂酸皂化生成硬脂酸盐作为乳化剂

  3. 保湿剂：甘油、丙二醇、1，3-丁二醇、山梨醇、聚乙二醇、神经酰胺

  4. 其他油性成分

  5. 水：雪花膏含水量为60%~80%

4.香波的组成

  1. 表面活性剂:去污、发泡

    • 阴离子表面活性剂

    • 非离子表面活性剂：增溶剂、分散剂、降低阴离子表面活性剂的刺激性，调节香波的粘度、稳定泡沫

    • 两性表面活性剂：用作增稠剂、调理剂

    • 阳离子表面活性剂：调理剂

  2. 添加剂

    • 去头屑止痒剂:水杨酸

    • 珠光剂：硬脂酸金属盐、乙二醇单硬脂酸

5.通用型香波

月桂醇硫酸酯钠盐

月桂醇醚硫酸钠盐

椰油基二乙醇酰胺

防腐剂、香精、着色剂

增稠剂（一般用NaCl）

去离子水

6.护发素

以阳离子表面活性剂（季铵盐类）为主要成分

7.牙膏应具有以下性能要求

  1. 具有适宜的摩擦力（摩擦剂）

  2. 具有良好的发泡性（阴离子型表面活性剂）

  3. 具有抑菌和防龋作用（氟化物）

  4. 有舒适的香味和口感（香精、甜味剂）

  5. 稳定性好（胶质原料）

  6. 安全性好

8.牙膏中的洗涤发泡剂即是牙膏中添加的表面活性剂，起作用是使牙膏具有去污、起泡的能力

防晒类化妆品

  1. 紫外线概述

名称 特征
UVC 被大气（臭氧层）吸收
UVB（晒红段） 穿透角质层和表皮
UVA（晒黑段） 穿透真皮

  1. 防晒剂

    化学吸收剂

    UVB吸收剂：二苯酮

    UVA吸收剂：二苯酮

  2. SPF：防晒化妆品中波紫外线防晒指数

  3. PFA：防晒化妆品长波紫外线防晒指数

保湿类化妆品:低于10%时，皮肤就会出现干燥、发皱、粗糙及脱屑等现象

保湿机理

  1. 在皮肤表面形成一层封闭式的油膜保护层，减少或阻止水分从皮肤表面蒸发
  2. 从大气中吸收水分使皮肤保持湿润

抗衰老化妆品

  • 日光中的紫外线辐射是环境因素中导致皮肤老化的主要因素，所以外源性老化又称光老化

  • 影响皮肤衰老的因素

    • 内在因素
    • 外在因素

美白类化妆品

  1. 黑素形成及原因

    紫外线是人类黑素细胞增殖和皮肤色素沉着增多的主要生理性刺激，也是使人类肤色变黑及生成色斑的主要途径和方式

  2. 美白祛斑化妆品的作用原理

    1. 抑制酪氨酸酶的活性、减少其产生及合成、加速其分解
    2. 防止黑素生成，破坏黑素细胞，抑制黑素颗粒的形成以及改变黑素颗粒的结构或还原黑素合成过程的中间体多巴醌
    3. 抑制黑素颗粒转移至角质形成细胞
    4. 加速角质形成细胞中的黑素向角质层转移及加快角质层脱落

  3. 美白活性物质

  1. 酪氨酸酶活性抑制剂：熊果苷

  2. 还原剂：维生素C

  3. 黑素运输阻断剂：壬二酸

  4. 皮肤剥落剂：果酸、亚油酸、亚麻酸

抗痤疮物质

  1. 维生素类
